Section 1: Architecture & DJL Integration

Leveraging Deep Java Library (DJL) for high-performance sequence modeling

Core Components

DJL Stack

  • • Engine: PyTorch (Native performance)
  • • Model: Sequential Block API
  • • Loss Function: Custom Pinball Loss (for quantiles)

Layer Stack

  • • Input Layer: (N, 30, 12)
  • • LSTM 1: 64 units (return sequences)
  • • LSTM 2: 32 units (final state)
  • • Dense: 3 outputs (Q5, Q50, Q95)

Why PyTorch Engine?

We will use the PyTorch engine via DJL because it has superior support for recurrent layers (LSTM) and allows for flexible custom loss functions like the Pinball Loss needed for quantile regression.
